Principais Comandos para Criar um Site

Comandos HTML Básicos

HTML é a linguagem de marcação usada para criar a estrutura de páginas web. Os comandos básicos incluem:

  • <html></html>: Define o início e o fim de um documento HTML.
  • <head></head>: Contém metadados, como o título da página e links para arquivos CSS.
  • <body></body>: Contém todo o conteúdo visível da página web.

Comandos CSS Básicos

CSS é usado para estilizar o conteúdo HTML. Alguns comandos básicos incluem:

  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte.
  • font-family: Define a fonte a ser usada.
